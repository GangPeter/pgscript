## 脚本名称

SpeedTest 净化增强

## 脚本简介

去除 SpeedTest 广告、拦截弹窗、修复布局、支持 PC 端｜移动端

### 1. 去除广告

-   横幅广告
-   限时活动

### 2. 拦截弹窗

-   拦截下载弹窗
-   拦截活动弹窗
-   拦截引导

### 3. 页面修复

-   去除页脚
-   去除顶栏
-   修复节点选择列表宽度
