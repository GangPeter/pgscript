## 脚本名称

JavLibrary 净化增强

## 脚本简介

去除 JavLibrary 广告、拦截弹窗、修复布局、支持 PC 端|移动端

## 脚本功能

### 1. 去除广告

-   去除 LOGO 右方广告
-   去除左侧菜单广告
-   去除左侧菜单游戏推广
-   去除页脚广告

### 2. 拦截弹窗

-   拦截成年同意弹窗

### 3. 页面修复

-   移除页脚
-   移除关于
