## 脚本名称

Bing 净化增强

## 脚本简介

去除 Bing 广告、拦截弹窗、修复布局、支持 PC 端｜移动端

## 下载链接

-   [Github](https://raw.githubusercontent.com/GangPeter/pgscript/refs/heads/main/dist/bing.js)
-   [GreasyFork](https://greasyfork.org/scripts/521192)

## 脚本功能

### 1. 去除广告

-   主页下载 APP 弹窗
-   主页搜索框推荐热搜词
-   搜索页面搜索结果广告
-   搜索页面更多结果广告
-   搜索页面搜索结果翻译下载 APP 按钮
-   搜索页面赚取积分推广二维码弹窗

### 2. 页面修复

-   去除主页、搜索页、航班页、学术页等页面页脚
-   去除每日测验题（防止鼠标靠近时显示）
-   去除每日一图信息
-   去除国际版今日图像、最近网页图像、历史今日、微软奖励、将 Bing 设置为首页等板块
-   去除右上角登陆、个人信息弹窗、奖励、现金、下载 Bing、手机版等按钮和二维码弹窗
-   修复去除搜索页面搜索结果广告后布局留白
-   修复去除板块后底部留白
